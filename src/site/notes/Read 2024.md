 Books I read in 2024
``` dataview
TABLE WITHOUT ID

("![coverImg|100](" + Cover + ")") as Cover,
("[[" + file.name + "|" + title + "]]") AS Title,
author,
rating,
finishedDate

FROM "50 Play/54 Media/54.05 Book Database"

WHERE contains(readingState, "read")
AND !contains(readingState, "unread")
AND !contains(readingState, "reading")
AND finishedDate.year = 2024

SORT finishedDate asc
```
